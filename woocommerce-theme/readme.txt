=== vanilla theme with woocommerce===
Contributors: psarge
Requires at least: 5.3
Tested up to: 5.6
Requires PHP: 5.6


== Description ==

Woocommerce-theme is a vanilla theme with basic capabilites that allow you to customize site using PHP.

Woocomerce plugin bootstrap is slightly customized on the row and column level to allow for boostrap within them to match the plugin code.
